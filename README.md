# Traffic Flow Prediction Using Neural Network

## 專案簡介

本專案以交通流量預測為主題，使用 Neural Network 建立交通資料預測模型，目標是根據歷史交通資料學習流量變化趨勢，並應用於智慧交通、道路壅塞預測與交通管理決策輔助。

This project focuses on traffic flow prediction using neural networks. It demonstrates how machine learning can be applied to intelligent transportation systems by learning traffic patterns from historical data and generating predictive results.

## 專案目標

* 建立交通流量預測模型，分析歷史交通資料與未來流量之間的關係。
* 練習時間序列或連續型資料的資料前處理方式。
* 使用 Neural Network 進行交通流量預測。
* 建立可延伸至智慧交通系統的預測模型流程。
* 將交通事故分析研究延伸至交通流量與道路風險預測應用。

## 技術重點

* Traffic Data Analysis：交通資料整理與趨勢分析。
* Data Preprocessing：缺失值處理、資料標準化、訓練資料建構。
* Neural Network Prediction：使用神經網路建立預測模型。
* Model Evaluation：使用 MAE、MSE、RMSE 等指標評估模型預測誤差。
* Visualization：視覺化實際交通流量與模型預測結果。

## 預計使用技術

* Python
* Pandas
* NumPy
* Scikit-learn
* TensorFlow / Keras 或 PyTorch
* Matplotlib
* Jupyter Notebook

## 建議專案流程

1. 載入交通流量資料集。
2. 檢查資料欄位、時間欄位與缺失值。
3. 進行資料清理與標準化。
4. 建立訓練資料與測試資料。
5. 建立 Neural Network 預測模型。
6. 訓練模型並觀察 loss 變化。
7. 使用測試資料進行交通流量預測。
8. 以 MAE、RMSE 等指標評估模型表現。
9. 繪製實際值與預測值比較圖。

## 專案價值

本專案可展示機器學習在智慧交通領域的應用能力，並與既有交通事故分析、GIS 空間分析與導航警示系統形成完整的智慧交通作品集。
